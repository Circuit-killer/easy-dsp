Python examples

The webapp includes several cool Python examples (for direction of arrival or speech detection among others). They are not from me, but come from Eric Bezzam and Robin Scheibler (LCAV, EPFL, Lausanne).

Vendors

  • d3;
  • jQuery;
  • Lodash;
  • Keymaster;
  • MicroPolar;
  • Recorderjs;
  • Rickshaw;
  • Cloud9 ACE.
